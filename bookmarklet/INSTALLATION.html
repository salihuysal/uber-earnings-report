2 So benutzt du es

  1. Melde dich im Uber Supplier Portal an
  2. Navigiere zur Earnings / Umsätze Seite
  3. Wichtig: Klicke auf das Zeitraum-Dropdown, damit alle Perioden sichtbar sind
  4. Klicke auf das 📊 Uber Report Lesezeichen
  5. Klicke auf ▶ Report generieren
  6. Warte, bis alle CSV-Dateien heruntergeladen wurden

3 Das bekommst du

📁
Eine CSV-Datei pro Mitarbeiter
📅
Alle Zeiträume automatisch
📊
Fare, Tips, Promotions, etc.
🔒
Keine Daten werden gesendet

📋 CSV in Excel öffnen

  1. Öffne Excel
  2. Klicke auf Datei → Öffnen
  3. Wähle die heruntergeladene CSV-Datei
  4. Die Daten werden automatisch als Tabelle angezeigt
💡 Hinweis: Die CSV-Dateien sind für deutsches Excel optimiert (Semikolon als Trennzeichen, Komma als Dezimalzeichen).
